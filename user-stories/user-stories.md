1. User Registration and Login

As a user, I want to be able to register and log into my account so that I can securely access the platform and manage my activities.

2. Property Management

As a host, I want to create, edit, and delete my property listings so that I can share my available spaces with potential guests.

3. Property Booking

As a guest, I want to search for and book properties so that I can reserve accommodations that meet my needs and preferences.

4. Payment Processing

As a guest, I want to make secure payments for my bookings so that I can confirm my reservation and receive proof of payment.

5. Review and Rating

As a guest, I want to leave a review and rate the property after my stay so that I can share my experience and help other users make informed decisions.

6. Messaging

As a guest or host, I want to send and receive messages so that I can communicate about booking details or ask questions before confirming a stay.

7. Admin Oversight

As an admin, I want to manage users, properties, and bookings so that I can maintain platform quality and handle issues effectively.
